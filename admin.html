💡 GitHub 토큰 발급 방법 (처음 한 번만)
  1. GitHub Fine-grained Token 페이지로 이동
  2. "Generate new token" 클릭
  3. Token name: quiz-admin 입력
  4. Expiration: "No expiration" 또는 원하는 기간 선택
  5. Repository access: "Only select repositories" 선택 → crcmc/quiz 선택
  6. Permissions → Repository permissions 섹션에서:
    • Contents: Read and write 선택
  7. 맨 아래 "Generate token" 클릭
  8. 표시된 토큰(github_pat_...)을 복사해서 아래에 붙여넣기
  9. ⚠️ 토큰은 한 번만 표시되니 안전한 곳에 백업해두세요!
