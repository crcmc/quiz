### 💡 좋은 이미지 고르는 팁
1. **"투명한 배경"** 필터를 사용하면 누끼 된 PNG 이미지만 볼 수 있어요
   - 왼쪽 필터 메뉴에서 **"투명한 배경"** 체크박스를 선택
2. **정면이나 측면에서 찍은** 곤충이 아이들이 식별하기 쉬워요
3. **밝고 선명한** 사진을 고르면 그림자 퀴즈에서도 잘 보여요
4. 이미지를 클릭 → **무료 다운로드** → **가장 작은 크기(1280x~)** 선택해도 충분해요

## 📝 다운로드 순서

1. 위 표에서 곤충 이름 옆 링크를 클릭
2. 마음에 드는 이미지를 선택 → 무료 다운로드
3. 다운로드한 파일의 이름을 표의 **파일명**으로 바꾸기 (예: `꿀벌-1234.jpg` → `bee.png`)
4. `images` 폴더에 넣기
5. 10개 모두 완성되면 `spring_insect_quiz.html` 실행!

💡 **JPG 파일을 받았을 경우**: 파일 확장자만 `.png`로 바꾸지 말고, 이름 자체를 `bee.png` 식으로 바꾸면 돼요. 브라우저는 확장자와 무관하게 이미지를 인식해요. (깔끔하게 하려면 [이미지 변환 사이트](https://www.iloveimg.com/ko/jpg-to-png)에서 PNG로 변환하세요.)

## 🎮 이미지 없이도 게임 가능!

사진을 아직 준비하지 못했어도 걱정 마세요! 이미지가 없으면 자동으로 **이모지**가 표시돼서 게임을 바로 플레이할 수 있어요. 나중에 사진을 추가하면 자동으로 사진으로 바뀝니다. 🎉
